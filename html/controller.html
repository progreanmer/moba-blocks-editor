This method find enemy which have lowest Hp in enemy list to be the target. The enemy list is list of enemy which in the vision of hero.

class controller.HeroItem¶

Bases: controller.Controller

Control hero about item. Example:

1.Hero item using. 2.Hero item selling. 3.Hero item buying. 4.Hero item checking type item.
buy_cheap_item(item_list)¶
Control hero to buy item which cheapest in item_list and hero can

buy. If hero can’t buy it, this method will not buy and operate next Instruction.

buy_expensive_item(item_list)¶
Control hero to buy item which most expensive in item_list and hero can

buy. If hero can’t buy it, this method will not buy and operate next Instruction.

buy_item_order(item_list)¶
Control hero to buy respectively item which in item_list and hero can

buy. If hero can’t buy it, this method will not buy and operate next Instruction.

sell_item(item)¶

Control hero to sell item which hero possess.
